总结

  1. iOS8 的 WKWebKit 不用再通过拦截 Request 来跟 js 交互了
  2. js 调用 native 通过 WKScriptMessageHandler
  3. native 调用 js 通过 WKWebView.evaluateJavaScript(javaScriptString:completionHandler:)
